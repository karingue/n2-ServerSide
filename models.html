O que são modelos de redes?

É a forma que é definido padrões para a comunicação de dados entre as redes de computadores.
Possuem diversas camadas para garantir a segurança e o funcionamento das redes.

Modelo OSI

Possui 7 camadas.

Camada 1 - Física

Lida com a transmissão de bits no meio físico, como cabos e sinais elétricos.

Camada 2 - Enlace de Dados

Controla o acesso ao meio, detecta erros e organiza dados em quadros.

Camada 3 - Rede

Roteia pacotes de dados entre redes diferentes e fornece endereçamento IP.

Camada 4 - Transporte

Gerencia a comunicação ponto a ponto, controla a confiabilidade e o fluxo de dados.

Camada 5 - Sessão

Estabelece, mantém e encerra sessões de comunicação entre aplicativos.

Camada 6 - Apresentação

Lida com a tradução e a criptografia de dados, garantindo a compatibilidade entre sistemas.

Camada 7 - Aplicação

Fornece interfaces para aplicativos de usuário final, como navegadores, e oferece serviços de rede.

Modelo TCP/IP

Possui 4 camadas.

Camada 1 - Acesso à Rede

Lida com o acesso direto ao meio físico, garantindo a transmissão confiável dos dados.

Camada 2 - Rede

Roteia pacotes de dados entre redes diferentes e fornece endereçamento IP.

Camada 3 - Transporte

Gerencia a comunicação ponto a ponto, controla a confiabilidade e o fluxo de dados.

Camada 4 - Aplicação

Fornece serviços de alto nível para aplicativos de usuário, como HTTP para navegação na web.
